# HLTV

CLI tool for parsing upcoming CS:GO matches using special filters.

Filters:

`--starred` - get list of matches marked by star  
`--today` - get only today matches  
`--team` - filter matches by given team

Usage:

```
$ ./hltv --starred  --team=Faze
            Liquid vs FaZe                0:00 ****
              FaZe vs MIBR                1:30 ***
```
